# 六边形地形编辑模式 — 完整设计文档

## 概述

在现有方形网格（Square Grid）地图编辑器基础上，新增**六边形网格（Hex Grid）**编辑模式。支持平顶/尖顶两种六边形朝向、六边形 Scene 视图可视化、六向邻居寻路、2D Tilemap 六边形渲染，与现有方形模式无缝切换、数据双向兼容。

---

## 一、坐标系设计

### 1.1 坐标系选择：轴向坐标（Axial Coordinates）

使用**轴向坐标 (q, r)** 作为六边形网格的核心坐标表示：

```
六边形网格中的每个六边形由 (q, r) 唯一标识：

     r=0   r=1   r=2   r=3
q=0   ○     ○     ○     ○
q=1     ○     ○     ○     ○
q=2   ○     ○     ○     ○
q=3     ○     ○     ○     ○
```

**选择轴向坐标的理由**：
- 用 `Vector2Int(q, r)` 表示，与现有数据模型完全兼容，无需改数据结构
- 六向邻居计算简单（6 个方向的偏移公式统一）
- Unity 内置 Hexagonal Tilemap 使用轴向坐标的变体
- 与立方坐标 (x, y, z) 可无损互转

### 1.2 轴向坐标 ↔ 立方坐标转换

```
立方坐标 (x, y, z) 满足约束: x + y + z = 0

轴向 → 立方:
  x = q
  z = r
  y = -x - z = -q - r

立方 → 轴向:
  q = x
  r = z
```

立方坐标用于距离计算和坐标圆整（pixel-to-hex 转换）。

### 1.3 数据存储映射

现有 `MapCellData` 不变，字段语义重新解释：

| 字段 | 方形网格含义 | 六边形网格含义 |
|------|-------------|---------------|
| `x` | grid column | 轴向 q 坐标 |
| `y` | grid row | 轴向 r 坐标 |
| `layer` | layer index | layer index（不变） |
| `terrainId` | terrain ID | terrain ID（不变） |

`MapUnitData` 同理：`x→q`, `y→r`。

## 二、六边形朝向与像素转换

### 2.1 两种朝向

支持两种六边形朝向，通过 `HexOrientation` 枚举控制：

| 朝向 | 说明 | 典型用途 |
|------|------|---------|
| **FlatTop（平顶）** | 六边形顶边水平，左右为尖角 | 策略游戏（文明系列）、战棋 |
| **PointyTop（尖顶）** | 六边形顶角朝上，上下为尖角 | 2D 俯视角 RPG、棋盘游戏 |

```
平顶 (FlatTop)      尖顶 (PointyTop)
  ⬡ ⬡ ⬡              ⬢
 ⬡ ⬡ ⬡               ⬢ ⬢
  ⬡ ⬡ ⬡              ⬢
```

### 2.2 六边形尺寸参数

定义 `cellSize` 为**六边形中心到顶点的距离**（即外接圆半径 R）：

```
           flat-top              pointy-top

    width = 2 * R          width = √3 * R
    height = √3 * R        height = 2 * R

    水平间距 W = 1.5 * R    水平间距 W = √3 * R
    垂直间距 H = √3 * R     垂直间距 H = 1.5 * R
```

特例：`cellSize = 1.0` 时 →
| 参数 | FlatTop | PointyTop |
|------|---------|-----------|
| width | 2.0 | 1.732 |
| height | 1.732 | 2.0 |
| 水平间距 | 1.5 | 1.732 |
| 垂直间距 | 1.732 | 1.5 |

### 2.3 六边形顶点计算

**平顶六边形** — 6 个顶点，从右顶点开始逆时针（角度 0°, 60°, 120°, 180°, 240°, 300°）：

```csharp
// flat-top corners (i = 0..5)
Vector3 center; // 六边形中心世界坐标
float angleDeg = 60f * i;
float angleRad = Mathf.PI / 180f * angleDeg;
Vector3 corner = center + new Vector3(
    size * Mathf.Cos(angleRad),
    0,
    size * Mathf.Sin(angleRad)
);
```

**尖顶六边形** — 6 个顶点，从右顶点开始，每个角再加 30°（角度 30°, 90°, 150°, 210°, 270°, 330°）：

```csharp
// pointy-top corners (i = 0..5)
float angleDeg = 60f * i + 30f;
// 其余同上
```

### 2.4 HexToWorld（轴向坐标 → 世界坐标）

**平顶**：
```csharp
// 3D 模式（XZ 平面，Y=0）
float x = cellSize * (1.5f * q);
float z = cellSize * (Mathf.Sqrt(3f) / 2f * q + Mathf.Sqrt(3f) * r);
Vector3 center = new Vector3(x, 0, z);
```

**尖顶**：
```csharp
float x = cellSize * (Mathf.Sqrt(3f) * q + Mathf.Sqrt(3f) / 2f * r);
float z = cellSize * (1.5f * r);
Vector3 center = new Vector3(x, 0, z);
```

### 2.5 WorldToHex（世界坐标 → 轴向坐标）

使用立方坐标圆整法（通用，平顶/尖顶通用）：

```csharp
// 1. 世界坐标 → 分数轴向坐标
float q, r;
if (flatTop)
{
    q = (Mathf.Sqrt(3f) / 3f * worldX - 1f / 3f * worldZ) / cellSize;
    r = (2f / 3f * worldZ) / cellSize;
}
else // pointyTop
{
    q = (2f / 3f * worldX) / cellSize;
    r = (-1f / 3f * worldX + Mathf.Sqrt(3f) / 3f * worldZ) / cellSize;
}

// 2. 分数轴向 → 立方坐标
float cx = q;
float cz = r;
float cy = -cx - cz;

// 3. 圆整立方坐标 (cube rounding)
int rx = Mathf.RoundToInt(cx);
int ry = Mathf.RoundToInt(cy);
int rz = Mathf.RoundToInt(cz);

float dx = Mathf.Abs(rx - cx);
float dy = Mathf.Abs(ry - cy);
float dz = Mathf.Abs(rz - cz);

if (dx > dy && dx > dz)
    rx = -ry - rz;
else if (dy > dz)
    ry = -rx - rz;

// 4. 立方 → 轴向
int resultQ = rx;
int resultR = rz;
```

### 2.6 六边形邻居

轴向坐标下，六边形有 **6 个邻居**，方向偏移向量统一：

```csharp
// 6 个轴向方向（顺序：右上、右、右下、左下、左、左上）
static readonly Vector2Int[] hexDirections = new Vector2Int[]
{
    new Vector2Int(+1, -1),  // 右上
    new Vector2Int(+1,  0),  // 右
    new Vector2Int( 0, +1),  // 右下
    new Vector2Int(-1, +1),  // 左下
    new Vector2Int(-1,  0),  // 左
    new Vector2Int( 0, -1),  // 左上
};

// 获取邻居的轴向坐标
Vector2Int neighborPos = new Vector2Int(q, r) + hexDirections[dir];
```

**注意**：这 6 个方向对平顶和尖顶都适用。区别仅体现在 WorldToHex 和 HexToWorld 的转换矩阵中。

### 2.7 六边形距离

```csharp
// 轴向距离（等于立方距离）
public static int HexDistance(Vector2Int a, Vector2Int b)
{
    // 转立方坐标
    int ax = a.x, az = a.y, ay = -ax - az;
    int bx = b.x, bz = b.y, by = -bx - bz;
    return (Mathf.Abs(ax - bx) + Mathf.Abs(ay - by) + Mathf.Abs(az - bz)) / 2;
}
```

## 四、HexGridUtils.cs — 六边形数学工具类（新建）

**路径**：`Assets/Scripts/Map/HexGridUtils.cs`

```csharp
using UnityEngine;

/// <summary>
/// 六边形网格数学工具。所有方法为静态，方形/六边形模式均可使用。
/// 使用轴向坐标 (q, r) 作为核心表示。
/// </summary>
public static class HexGridUtils
{
    // ==================== 朝向枚举 ====================
    public enum HexOrientation { FlatTop, PointyTop }

    // ==================== 常量 ====================
    private const float SQRT3 = 1.73205080757f;
    private const float SQRT3_OVER_2 = 0.86602540378f;
    private const float SQRT3_OVER_3 = 0.57735026919f;
    private const float ONE_THIRD = 0.33333333333f;
    private const float TWO_THIRDS = 0.66666666667f;
    private const float ONE_POINT_FIVE = 1.5f;

    // 6 个轴向邻居方向
    public static readonly Vector2Int[] HexNeighbors = new Vector2Int[]
    {
        new Vector2Int(+1, -1), // 0: 右上 (NE)
        new Vector2Int(+1,  0), // 1: 右   (E)
        new Vector2Int( 0, +1), // 2: 右下 (SE)
        new Vector2Int(-1, +1), // 3: 左下 (SW)
        new Vector2Int(-1,  0), // 4: 左   (W)
        new Vector2Int( 0, -1), // 5: 左上 (NW)
    };

    // ==================== 坐标转换 ====================

    /// <summary>轴向坐标 → 3D 世界坐标（中心点）</summary>
    public static Vector3 HexToWorld3D(int q, int r, float cellSize, HexOrientation orientation)
    {
        if (orientation == HexOrientation.FlatTop)
        {
            float x = cellSize * (1.5f * q);
            float z = cellSize * (SQRT3_OVER_2 * q + SQRT3 * r);
            return new Vector3(x, 0, z);
        }
        else // PointyTop
        {
            float x = cellSize * (SQRT3 * q + SQRT3_OVER_2 * r);
            float z = cellSize * (1.5f * r);
            return new Vector3(x, 0, z);
        }
    }

    /// <summary>轴向坐标 → 3D 世界坐标（Vector2Int 重载）</summary>
    public static Vector3 HexToWorld3D(Vector2Int hex, float cellSize, HexOrientation orientation)
        => HexToWorld3D(hex.x, hex.y, cellSize, orientation);

    /// <summary>轴向坐标 → 2D 世界坐标（XY 平面）</summary>
    public static Vector3 HexToWorld2D(int q, int r, float cellSize, HexOrientation orientation)
    {
        if (orientation == HexOrientation.FlatTop)
        {
            float x = cellSize * (1.5f * q);
            float y = cellSize * (SQRT3_OVER_2 * q + SQRT3 * r);
            return new Vector3(x, y, 0);
        }
        else // PointyTop
        {
            float x = cellSize * (SQRT3 * q + SQRT3_OVER_2 * r);
            float y = cellSize * (1.5f * r);
            return new Vector3(x, y, 0);
        }
    }

    /// <summary>轴向坐标 → 2D 世界坐标（Vector2Int 重载）</summary>
    public static Vector3 HexToWorld2D(Vector2Int hex, float cellSize, HexOrientation orientation)
        => HexToWorld2D(hex.x, hex.y, cellSize, orientation);

    /// <summary>世界坐标 → 轴向坐标（3D，XZ 平面）</summary>
    public static Vector2Int WorldToHex3D(Vector3 worldPos, float cellSize, HexOrientation orientation)
    {
        float q, r;

        if (orientation == HexOrientation.FlatTop)
        {
            q = (SQRT3_OVER_3 * worldPos.x - ONE_THIRD * worldPos.z) / cellSize;
            r = (TWO_THIRDS * worldPos.z) / cellSize;
        }
        else
        {
            q = (TWO_THIRDS * worldPos.x) / cellSize;
            r = (-ONE_THIRD * worldPos.x + SQRT3_OVER_3 * worldPos.z) / cellSize;
        }

        return CubeRound(q, r);
    }

    /// <summary>世界坐标 → 轴向坐标（2D，XY 平面）</summary>
    public static Vector2Int WorldToHex2D(Vector3 worldPos, float cellSize, HexOrientation orientation)
    {
        float q, r;

        if (orientation == HexOrientation.FlatTop)
        {
            q = (SQRT3_OVER_3 * worldPos.x - ONE_THIRD * worldPos.y) / cellSize;
            r = (TWO_THIRDS * worldPos.y) / cellSize;
        }
        else
        {
            q = (TWO_THIRDS * worldPos.x) / cellSize;
            r = (-ONE_THIRD * worldPos.x + SQRT3_OVER_3 * worldPos.y) / cellSize;
        }

        return CubeRound(q, r);
    }

    // ==================== 立方坐标圆整 ====================

    private static Vector2Int CubeRound(float q, float r)
    {
        float cx = q;
        float cz = r;
        float cy = -cx - cz;

        int rx = Mathf.RoundToInt(cx);
        int ry = Mathf.RoundToInt(cy);
        int rz = Mathf.RoundToInt(cz);

        float dx = Mathf.Abs(rx - cx);
        float dy = Mathf.Abs(ry - cy);
        float dz = Mathf.Abs(rz - cz);

        if (dx > dy && dx > dz)
            rx = -ry - rz;
        else if (dy > dz)
            ry = -rx - rz;

        return new Vector2Int(rx, rz);
    }

    // ==================== 六边形顶点 ====================

    /// <summary>获取六边形 6 个顶点的世界坐标（3D，XZ 平面）</summary>
    public static Vector3[] GetHexCorners3D(Vector3 center, float cellSize, HexOrientation orientation)
    {
        Vector3[] corners = new Vector3[6];
        float startAngle = (orientation == HexOrientation.FlatTop) ? 0f : 30f;

        for (int i = 0; i < 6; i++)
        {
            float angleDeg = 60f * i + startAngle;
            float angleRad = angleDeg * Mathf.Deg2Rad;
            corners[i] = center + new Vector3(
                cellSize * Mathf.Cos(angleRad),
                0,
                cellSize * Mathf.Sin(angleRad)
            );
        }
        return corners;
    }

    /// <summary>获取六边形 6 个顶点的世界坐标（2D，XY 平面）</summary>
    public static Vector3[] GetHexCorners2D(Vector3 center, float cellSize, HexOrientation orientation)
    {
        Vector3[] corners = new Vector3[6];
        float startAngle = (orientation == HexOrientation.FlatTop) ? 0f : 30f;

        for (int i = 0; i < 6; i++)
        {
            float angleDeg = 60f * i + startAngle;
            float angleRad = angleDeg * Mathf.Deg2Rad;
            corners[i] = center + new Vector3(
                cellSize * Mathf.Cos(angleRad),
                cellSize * Mathf.Sin(angleRad),
                0
            );
        }
        return corners;
    }

    // ==================== 距离与邻接 ====================

    /// <summary>两个六边形之间的轴向距离</summary>
    public static int HexDistance(Vector2Int a, Vector2Int b)
    {
        int ax = a.x, az = a.y, ay = -ax - az;
        int bx = b.x, bz = b.y, by = -bx - bz;
        return (Mathf.Abs(ax - bx) + Mathf.Abs(ay - by) + Mathf.Abs(az - bz)) / 2;
    }

    /// <summary>判断相邻（距离为 1）</summary>
    public static bool IsAdjacent(Vector2Int a, Vector2Int b)
        => HexDistance(a, b) == 1;

    /// <summary>获取某六边形的所有邻居坐标</summary>
    public static List<Vector2Int> GetNeighbors(Vector2Int hex)
    {
        var result = new List<Vector2Int>(6);
        foreach (var dir in HexNeighbors)
            result.Add(hex + dir);
        return result;
    }

    /// <summary>获取某六边形的所有邻居坐标（带边界检查）</summary>
    public static List<Vector2Int> GetNeighborsInBounds(Vector2Int hex, int mapWidth, int mapHeight)
    {
        var result = new List<Vector2Int>(6);
        foreach (var dir in HexNeighbors)
        {
            Vector2Int neighbor = hex + dir;
            if (neighbor.x >= 0 && neighbor.x < mapWidth && neighbor.y >= 0 && neighbor.y < mapHeight)
                result.Add(neighbor);
        }
        return result;
    }

    // ==================== 六边形环与范围 ====================

    /// <summary>获取距离中心 hex 为 range 范围内的所有六边形</summary>
    public static List<Vector2Int> GetHexagonsInRange(Vector2Int center, int range)
    {
        var result = new List<Vector2Int>();
        for (int dq = -range; dq <= range; dq++)
        {
            int minR = Mathf.Max(-range, -dq - range);
            int maxR = Mathf.Min(range, -dq + range);
            for (int dr = minR; dr <= maxR; dr++)
            {
                result.Add(new Vector2Int(center.x + dq, center.y + dr));
            }
        }
        return result;
    }

    // ==================== 视线（Line of Sight） ====================

    /// <summary>两个六边形之间的直线路径（Bresenham 式线性插值）</summary>
    public static List<Vector2Int> HexLine(Vector2Int from, Vector2Int to)
    {
        int dist = HexDistance(from, to);
        var result = new List<Vector2Int>();

        float step = 1f / Mathf.Max(dist, 1);
        for (int i = 0; i <= dist; i++)
        {
            float t = step * i;
            // 使用立方坐标插值
            float ax = from.x, az = from.y, ay = -ax - az;
            float bx = to.x, bz = to.y, by = -bx - bz;

            float lx = ax + (bx - ax) * t;
            float ly = ay + (by - ay) * t;
            float lz = az + (bz - az) * t;

            result.Add(CubeRound(lx, lz));
        }
        return result;
    }

    // ==================== 地图边界世界坐标 ====================

    /// <summary>计算六边形地图在世界空间中的大致包围矩形</summary>
    public static (Vector3 min, Vector3 max) GetMapBoundsWorld(int mapWidth, int mapHeight,
        float cellSize, HexOrientation orientation)
    {
        // 采样四个角来计算包围盒
        Vector3[] sampleHexes = new Vector3[]
        {
            HexToWorld3D(0, 0, cellSize, orientation),
            HexToWorld3D(mapWidth - 1, 0, cellSize, orientation),
            HexToWorld3D(0, mapHeight - 1, cellSize, orientation),
            HexToWorld3D(mapWidth - 1, mapHeight - 1, cellSize, orientation),
        };

        Vector3 min = sampleHexes[0], max = sampleHexes[0];
        foreach (var p in sampleHexes)
        {
            min = Vector3.Min(min, p);
            max = Vector3.Max(max, p);
        }

        float padding = cellSize * 1.2f;
        min -= new Vector3(padding, 0, padding);
        max += new Vector3(padding, 0, padding);

        return (min, max);
    }

    /// <summary>计算六边形地图在 2D 空间中的大致包围矩形</summary>
    public static (Vector3 min, Vector3 max) GetMapBoundsWorld2D(int mapWidth, int mapHeight,
        float cellSize, HexOrientation orientation)
    {
        Vector3[] sampleHexes = new Vector3[]
        {
            HexToWorld2D(0, 0, cellSize, orientation),
            HexToWorld2D(mapWidth - 1, 0, cellSize, orientation),
            HexToWorld2D(0, mapHeight - 1, cellSize, orientation),
            HexToWorld2D(mapWidth - 1, mapHeight - 1, cellSize, orientation),
        };

        Vector3 min = sampleHexes[0], max = sampleHexes[0];
        foreach (var p in sampleHexes)
        {
            min = Vector3.Min(min, p);
            max = Vector3.Max(max, p);
        }

        float padding = cellSize * 1.2f;
        min -= new Vector3(padding, padding, 0);
        max += new Vector3(padding, padding, 0);

        return (min, max);
    }
}
```

## 五、数据结构修改

### 5.1 MapJsonData.cs — 增加网格形状字段

在现有类中增加两个字段：

```csharp
[Serializable]
public class MapJsonData
{
    // ... 现有字段保持不变 ...

    /// <summary>网格形状类型："Square" 或 "Hex"（默认 "Square" 向后兼容）</summary>
    public string gridShape = "Square";

    /// <summary>六边形朝向（仅 gridShape == "Hex" 时有效）："FlatTop" 或 "PointyTop"</summary>
    public string hexOrientation = "FlatTop";

    // ... cellDatas, unitDatas 等保持不变 ...
}
```

**向后兼容策略**：旧 JSON 文件没有 `gridShape` 字段时，`JsonUtility` 自动反序列化为 null → 编辑器检测到 null 时视为 "Square"，行为不变。

### 5.2 编辑器内部数据结构

`MapEditorWindow.cs` 增加两个全局状态字段：

```csharp
public enum GridShape
{
    Square,
    HexFlatTop,
    HexPointyTop
}

private GridShape _gridShape = GridShape.Square;
// 注意：_cellSize 在方形和六边形模式下共用，语义不同：
//   方形：格子边长
//   六边形：外接圆半径
// 切换形状时 _cellSize 保持不变（用户可手动调整）
```

`_allLayers` 和 `_allUnitLayers` 的 `Dictionary<Vector2Int, string>` 结构不变，六边形模式下 `Vector2Int.x = q, Vector2Int.y = r`。

### 5.3 坐标边界

- **方形网格**：`0 <= x < mapWidth`, `0 <= y < mapHeight`
- **六边形网格**：`0 <= q < mapWidth`, `0 <= r < mapHeight`（轴向坐标的平行四边形区域）
- `mapWidth` 和 `mapHeight` 对六边形网格的含义：网格在 q 轴和 r 轴上的跨度

```csharp

{
```

```csharp
    if (e.type == EventType.MouseDrag && e.button == 0 && _isPainting)
    {
        if (hex != _lastPaintedCell)
        {
            _lastPaintedCell = hex;
            BeginUndoOp();
            PaintCell(hex);
            EndUndoOp();
        }
        e.Use();
        return;
    }

    // 左键抬起：结束操作
    if (e.type == EventType.MouseUp && e.button == 0)
    {
        _isPainting = false;
        e.Use();
    }
```

### 8.2 六边形矩形填充

在六边形网格中，"矩形填充"实际上是**菱形区域填充**，填充所有在两个轴向坐标范围之间的六边形：

```csharp
/// <summary>六边形菱形区域填充</summary>
private void HexFillRhombus(Vector2Int a, Vector2Int b, string terrainId)
{
    int q0 = Mathf.Clamp(Mathf.Min(a.x, b.x), 0, _mapWidth - 1);
    int q1 = Mathf.Clamp(Mathf.Max(a.x, b.x), 0, _mapWidth - 1);
    int r0 = Mathf.Clamp(Mathf.Min(a.y, b.y), 0, _mapHeight - 1);
    int r1 = Mathf.Clamp(Mathf.Max(a.y, b.y), 0, _mapHeight - 1);

    // 菱形区域：所有满足 q0..q1, r0..r1 的六边形
    BeginUndoOp();
    for (int q = q0; q <= q1; q++)
        for (int r = r0; r <= r1; r++)
        {
            RecordChange(new Vector2Int(q, r));
            _mapGrid[new Vector2Int(q, r)] = terrainId;
        }
    EndUndoOp();
    Repaint();
}
```

### 8.3 六边形洪水填充（Flood Fill）

使用六向邻居进行 BFS 洪水填充：

```csharp
/// <summary>六边形洪水填充</summary>
private void HexFloodFill(Vector2Int start, string fillTerrainId)
{
    // 起点必须是已有地形
    if (!_mapGrid.TryGetValue(start, out string targetId)) return;
    if (targetId == fillTerrainId) return; // 同种地形不填充

    var queue = new Queue<Vector2Int>();
    var visited = new HashSet<Vector2Int>();
    queue.Enqueue(start);
    visited.Add(start);

    BeginUndoOp();

    while (queue.Count > 0)
    {
        Vector2Int current = queue.Dequeue();

        if (_mapGrid.TryGetValue(current, out string id) && id == targetId)
        {
            RecordChange(current);
            _mapGrid[current] = fillTerrainId;

            // 六向邻居 BFS
            var neighbors = HexGridUtils.GetNeighborsInBounds(current, _mapWidth, _mapHeight);
            foreach (var neighbor in neighbors)
            {
                if (visited.Add(neighbor))
                    queue.Enqueue(neighbor);
            }
        }
    }

    EndUndoOp();
    Repaint();
}
```

## 十、A* 寻路扩展（六边形模式）

### 10.1 MapPathfinding.cs 修改

在 `FindPath()` 方法中增加六边形分支：

```csharp
/// <summary>查找从起点到终点的最短路径</summary>
/// <param name="gridShape">网格形状："Square" 或 "Hex"</param>
/// <param name="hexOrientation">六边形朝向（Hex 模式时有效）</param>
public static List<Vector2Int> FindPath(
    int startX, int startY,
    int endX, int endY,
    string unitId,
    MapJsonData mapData,
    bool allowDiagonal = false,
    string gridShape = "Square",
    string hexOrientation = null)
{
    if (mapData == null) return new List<Vector2Int>();
    if (!IsInBounds(startX, startY, mapData) || !IsInBounds(endX, endY, mapData))
        return new List<Vector2Int>();
    if (!IsPassable(endX, endY, unitId, mapData))
        return new List<Vector2Int>();

    Vector2Int start = new Vector2Int(startX, startY);
    Vector2Int end = new Vector2Int(endX, endY);
    if (start == end) return new List<Vector2Int> { start };

    // ... heap 和 nodeCache 创建（不变）...

    while (openSet.Count > 0)
    {
        // ... 弹出最小代价节点 ...

            ? GetSquareNeighbors(current, allowDiagonal)
            : GetHexNeighbors(current);

        {
            if (!IsInBounds(neighbor.x, neighbor.y, mapData)) continue;
            if (closedSet.Contains(neighbor)) continue;
            if (!IsPassable(neighbor.x, neighbor.y, unitId, mapData)) continue;

            // 六边形移动不需要穿角检测（六边形没有对角移动）
            if (gridShape == "Square" && allowDiagonal)
            {
                // 现有穿角检测逻辑...
            }

            Node neighborNode = GetOrCreateNode(neighbor.x, neighbor.y, nodeCache);
            float moveCost = 1f; // 六边形每个邻居距离相同

            // 考虑地形额外移动消耗
            var terrain = GameMapGlobal.GetTerrainAt(neighbor.x, neighbor.y, 0, mapData);
            if (terrain != null)
            {
                float terrainCost = terrain.GetFloat("移动消耗");
                if (terrainCost > 0) moveCost += terrainCost;
            }

            float newGCost = current.gCost + moveCost;
            if (newGCost < neighborNode.gCost)
            {
                neighborNode.gCost = newGCost;
                neighborNode.hCost = Heuristic(neighbor.x, neighbor.y, endX, endY, gridShape);
                neighborNode.parent = current;

                if (!openSet.Contains(neighborNode))
                    openSet.Add(neighborNode);
                else
                    openSet.UpdateItem(neighborNode);
            }
        }
    }

    return new List<Vector2Int>();
}

    foreach (var dir in HexGridUtils.HexNeighbors)
        result.Add(current.pos + dir);
    return result;
}

private static List<Vector2Int> GetSquareNeighbors(Node current, bool allowDiagonal) { /* 现有逻辑 */ }

/// <summary>启发函数（六边形用六边形距离，方形用曼哈顿/切比雪夫）</summary>
private static float Heuristic(int x1, int y1, int x2, int y2, string gridShape)
{
    if (gridShape == "Square")
        return 10 * (Mathf.Abs(x1 - x2) + Mathf.Abs(y1 - y2)); // 曼哈顿

        new Vector2Int(x2, y2));
```

```csharp
    var data = JsonUtility.FromJson<MapJsonData>(mapJson.text);

    Grid grid = GetComponent<Grid>();
    if (grid == null) grid = gameObject.AddComponent<Grid>();

    if (data.gridShape == "Hex")
    {
        grid.cellLayout = GridLayout.CellLayout.Hexagon;
        // cellSize 使用 Unity hex tilemap 的默认值，或从 data 读取
    }
    else
    {
        grid.cellLayout = GridLayout.CellLayout.Rectangle;
    }

    Tilemap tilemap = GetComponentInChildren<Tilemap>();
    if (tilemap == null)
    {
        var go = new GameObject("Tilemap");
        go.transform.SetParent(transform);
        tilemap = go.AddComponent<Tilemap>();
        go.AddComponent<TilemapRenderer>();
    }

    foreach (var cell in data.cellDatas)
    {
        var terrain = config.GetTerrainById(cell.terrainId);
        if (terrain?.terrainSprite == null) continue;

        Tile tile = ScriptableObject.CreateInstance<Tile>();
        tile.sprite = terrain.terrainSprite;
        tile.color = terrain.defaultType switch { /* 按类型染色 */ };

        tilemap.SetTile(new Vector3Int(cell.x, cell.y, 0), tile);
    }
}
```

## 十六、核心设计原则总结

1. **最小侵入**：不修改现有坐标字段 `x/y`，仅增加 `gridShape`/`hexOrientation` 两个字符串字段
2. **轴向坐标为王**：全系统统一使用轴向坐标 (q, r)，仅在 Scene 渲染和鼠标交互时做 World ↔ Hex 转换
3. **数学集中**：所有六边形数学逻辑集中在 `HexGridUtils.cs` 静态类，其他文件通过调用该工具类实现功能
4. **向后兼容**：现有方形网格功能不受影响；旧 JSON 无 `gridShape` 字段时自动按方形加载
5. **共享数据模型**：`Dictionary<Vector2Int, string>` 的数据结构不变，方形模式下 x=列/y=行，六边形模式下 x=q/y=r
6. **面向扩展**：`HexOrientation` 枚举预留扩展点；后续可增加更多六边形形状（环形、矩形裁剪、六边形菱形范围等）

### P1：平行四边形状地图不够直观

**问题**：轴向坐标 + 矩形边界 = 平行四边形地图。用户对六边形地图的直觉是"蜂窝状"或"近似矩形"。

```
当前方案（平行四边形）         用户期望的矩形六边形地图
  ○ ○ ○ ○ ○                    ○ ○ ○ ○ ○
 ○ ○ ○ ○ ○ ○                  ○ ○ ○ ○ ○ ○
○ ○ ○ ○ ○ ○ ○                ○ ○ ○ ○ ○ ○
 ○ ○ ○ ○ ○ ○                  ○ ○ ○ ○ ○ ○
```

**补充方案**：分阶段支持三种地图形状

| 形状 | 说明 | 实现难度 | 优先级 |
|------|------|---------|--------|
| **平行四边形** (Parallelogram) | 0≤q<W, 0≤r<H，首版默认 | 低（已完成设计） | P0 |
| **偏移矩形** (Offset Rectangle) | 使用 odd-r/even-r 偏移坐标，row 奇数行较短 | 中 | P1 |
| **六边形** (Hexagon Shape) | 以中心为原点，按半径 range 展开 | 中 | P2 |

**偏移矩形实现思路**（P1 优先级）：
```csharp
// odd-r offset 坐标，转换到轴向再存储
// 列数 c ∈ [0, mapWidth), 行数 r ∈ [0, mapHeight)
// 奇数行少一列
public static bool IsValidOffsetRect(int col, int row, int mapWidth, int mapHeight)
{
    if (row < 0 || row >= mapHeight) return false;
    if (col < 0) return false;
    // 奇数行：列比偶数行少 1
    if (row % 2 == 1)
        return col < mapWidth - 1;
    else
        return col < mapWidth;
}
```

首版先实现平行四边形，文档和 UI 明确告知用户当前形状类型。

### P5：2D Tilemap CellSize 映射不一致

**问题**：设计中将编辑器 `cellSize`（外接圆半径）直接转为 Unity Hex Grid 的 `cellSize` 向量，但两者的物理含义不同：

| 含义 | 编辑器 cellSize | Unity Grid.cellSize |
|------|----------------|---------------------|
| 平顶 | 外接圆半径 R | (水平间距, 垂直间距) = (1.5R, √3·R) |
| 尖顶 | 外接圆半径 R | (水平间距, 垂直间距) = (√3·R, 1.5R) |

**补充方案**：在 JSON 中存储 `hexCellSize`（外接圆半径），在 Tilemap 初始化时显式转换：

```csharp
// MapJsonData 新增字段
public float hexCellSize = 1.0f; // 六边形外接圆半径（仅 gridShape=="Hex" 时有效）

// MapEditor2DSceneManager.Setup() 中：
float R = _owner.CellSize; // 编辑器外接圆半径
if (_owner.GridShape == GridShape.HexFlatTop)
    grid.cellSize = new Vector3(R * 1.5f, R * HexGridUtils.SQRT3, 1);
else
    grid.cellSize = new Vector3(R * HexGridUtils.SQRT3, R * 1.5f, 1);
```

同时需要在 JSON 中加上 `hexCellSize` 字段，保证跨机器一致。

### P9：缺失"六边形范围填充"工具

**问题**：六边形网格的一个自然操作是"以某格为中心，填充 range=N 的六边形区域"（六边形刷）。当前设计只有菱形填充（对应方形的矩形填充），缺少真正的六边形范围填充。

**补充方案**：增加六边形范围填充模式，利用 `HexGridUtils.GetHexagonsInRange()`：

```csharp
/// <summary>六边形范围填充：以 center 为中心，填充 range 范围内的所有六边形</summary>
private void HexRangeFill(Vector2Int center, int range, string terrainId)
{
    var cells = HexGridUtils.GetHexagonsInRange(center, range);

    BeginUndoOp();
    foreach (var hex in cells)
    {
        if (hex.x >= 0 && hex.x < _mapWidth && hex.y >= 0 && hex.y < _mapHeight)
        {
            RecordChange(hex);
            _mapGrid[hex] = terrainId;
        }
    }
    EndUndoOp();
    Repaint();
}
```

在编辑器的编辑模式枚举中增加 `HexRangeFill`：

```csharp
// 仅六边形模式可用的编辑模式
private bool _showHexTools = false; // 六边形专用工具折叠面板
```

范围由编辑器 UI 的 slider 控制（range = 0~10）。

### P12：Unity Hex Tilemap CellSwizzle 细节

**问题**：9.1 节提到 `grid.cellSwizzle = GridLayout.CellSwizzle.XYZ` 但未说明不同朝向的配置。

**补充方案**：Unity Hex Tilemap 完整配置表：

| 朝向 | CellLayout | CellSwizzle | 说明 |
|------|-----------|-------------|------|
| 平顶 (FlatTop) | `Hexagon` | `XYZ` (默认) | X 轴为锯齿方向 |
| 尖顶 (PointyTop) | 需要自定义 | — | **Unity 原生 Hex Grid 仅支持平顶** |

**重要发现**：Unity 的 `GridLayout.CellLayout.Hexagon` **仅支持平顶布局**。尖顶六边形不能用 Unity 内置 Hex Tilemap 实现。

**应对方案**：
1. 2D 尖顶模式改用自定义 `Handles` 渲染（与 3D 模式相同），放弃 Unity Tilemap 加速
2. 或限制 2D 六边形模式仅支持平顶

建议采用方案 2，减少开发复杂度。如果用户需要尖顶六边形，仍可用 3D 模式编辑。文档和 UI 需明确说明此限制。

### P16：WorldToHex 在六边形边缘的点击归属

**问题**：在相邻六边形的边界处点击，cube rounding 算法会将其归入两个六边形之一。边界处的归属取决于浮点精度，理论上会在接近正中间时产生"随机"归属，属于六边形网格的普遍现象。

**补充方案**：无需特别处理。cube rounding 是业界标准方法。如果用户对边界归属不满，可以放大 cellSize 提高编辑精度。文档中提及此行为即可。
