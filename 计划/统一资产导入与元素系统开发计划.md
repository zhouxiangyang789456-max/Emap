# 统一资产导入与元素系统 — 开发计划

## 概述

将当前分散的「地形创建」「单位创建」合并为统一的资产管理系统，新增**元素（Element）**类型支持非地形/非单位预制体导入，所有资产类型共享「拖入预制体→自动创建配置→立即可用」的标准化流程。同时为所有资产类型（地形/单位/元素）增加预制体支持。

**关联设计文档**：`统一资产导入与元素系统设计文档.md`
**前置依赖**：六边形编辑模式应先于本功能完成

---

## 验收标准

1. 资产管理器（MapAssetManagerWindow）三标签页正常运作
2. 拖入 prefab 能自动创建地形/单位/元素配置
3. 元素层可正常添加、切换、编辑（笔刷/橡皮/填充）
4. 元素放置规则（Free/RequiresTerrain/EmptyOnly/Unique）全部生效
5. 3D Scene 视图中可显示地形和元素的 prefab 预览
6. 2D 模式下元素的 prefab 正确实例化（含正确的分层 Z 轴排序）
7. 运行时地图生成正确实例化地形和元素的 prefab
8. JSON 保存/加载 elementDatas 数据往返一致
9. 撤销/重做正确支持元素层操作
10. 旧 JSON 和旧配置向后兼容，无崩溃
11. 删除 ElementConfig 后加载地图弹出悬空引用警告
12. 100×100 地图 + 1000 个元素 prefab 实例时编辑性能可接受
