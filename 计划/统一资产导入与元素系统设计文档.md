# 统一资产导入与元素系统 — 设计文档

## 概述

将当前分散的「地形创建」「单位创建」合并为统一的资产管理系统，新增**元素（Element）**类型支持非地形/非单位预制体导入，所有资产类型共享「拖入预制体→自动创建配置→立即可用」的标准化流程。同时为所有资产类型（地形/单位/元素）增加预制体支持，解决插件拖入项目后预制体资产难以管理和使用的问题。

---

## 一、现状分析与问题

### 1.1 当前资产创建方式（分散）

| 资产类型 | 创建方式 | 存储形式 | 预制体支持 |
|---------|---------|---------|-----------|
| **地形** (TerrainResource) | 编辑器内表单填写 → 加入 MapResourceConfig 列表 | MapResourceConfig.asset 内的 List 元素 | 无（仅 Sprite） |
| **单位** (UnitConfig) | SaveFilePanel → CreateAsset → 独立 .asset 文件 | 独立 .asset 文件 | 有字段但从未实例化 |

### 1.2 核心痛点

| # | 痛点 | 影响 |
|---|------|------|
| P1 | **预制体用不上**：UnitConfig.prefab 字段从创建到渲染、运行时，从未被实例化 | 用户拖入了 prefab 但完全不起作用 |
| P2 | **地形不支持预制体**：TerrainResource 只有 Sprite，3D 项目中无法用 3D 模型做地形 | 只能做 2D/2.5D 地形 |
| P3 | **非地形元素无处可放**：树木、岩石、建筑装饰物等无法作为地图元素导入 | 缺少"场景装饰层"概念 |
| P4 | **创建流程分散**：三种事物的管理入口不同（菜单/Project 右键/编辑器按钮），用户困惑 | 学习成本高 |
| P5 | **批量导入困难**：项目中有 20 个 prefab 想作为地图元素，需逐个手动创建配置 | 重复劳动 |

### 1.3 当前编辑器中 unit/prefab 使用情况

```
UnitConfig.prefab 字段：
  - MapEditorWindow.cs:542  仅显示 ObjectField（只读），从未实例化
  - MapEditorWindow.cs:926  单位渲染用蓝色球体 Handles.SphereHandleCap
  - MapEditor2DSceneManager:314  单位渲染用蓝色圆盘 Handles.DrawSolidDisc
  - RuntimeMapGenerator.cs  完全不处理单位数据

TerrainResource.terrainSprite：
  - MapEditorWindow.cs:887  3D 模式用 Handles.CubeHandleCap 色块
  - MapEditor2DSceneManager:157  2D 模式转 Unity Tile
  - RuntimeMapGenerator.cs:108  运行时创建 SpriteRenderer

结论：编辑器和运行时都没有使用 GameObject prefab。
```

```csharp
// Assets/Scripts/Map/ElementConfig.cs

using System.Collections.Generic;
using UnityEngine;

/// <summary>
/// 元素配置：非地形/非单位的可放置装饰物（树木、岩石、建筑等）。
/// 每个元素对应一个 .asset 文件。
/// </summary>
[CreateAssetMenu(fileName = "ElementConfig", menuName = "地图编辑器/元素配置")]
public class ElementConfig : ScriptableObject
{
    [Header("基础信息")]
    public string elementId;           // 唯一 ID（如 "oak_tree"）
    public string elementName;         // 显示名称（如 "橡树"）
    public GameObject prefab;          // 预制体（运行时实例化）

    [Header("显示")]
    public Sprite previewSprite;       // 2D 模式/列表中显示的精灵（可选，为空则用 prefab 缩略图）

    [Header("放置规则")]
    public ElementPlaceRule placeRule = ElementPlaceRule.Free; // 放置规则
    public bool canStack = false;      // 是否允许同一格放多个元素

    [Header("碰撞")]
    public bool generateCollider = true;      // 运行时是否生成碰撞体
    public bool blocksMovement = false;       // 是否阻挡单位移动

    [Header("自定义属性")]
    public List<CustomProperty> customProperties = new List<CustomProperty>();

    // —— 便捷取值（与 TerrainResource 一致） ——
    public int GetInt(string propName) { /* 同 TerrainResource */ }
    public float GetFloat(string propName) { /* 同 TerrainResource */ }
    public bool GetBool(string propName) { /* 同 TerrainResource */ }
}

/// <summary>元素放置规则</summary>
public enum ElementPlaceRule
{
    Free,           // 可放在任何格子（包括有地形的）
    RequiresTerrain, // 需要格子已有地形
    EmptyOnly,      // 只能放在空格子
    Unique          // 每个元素 ID 全局唯一（如 Boss 标记）
}
```

### 5.3 面板核心操作

#### 操作 A：新建资产

```
点击 [+ 新建] → 弹出选择框:
  ○ 地形       → 创建 TerrainResource
  ○ 单位       → 创建 UnitConfig (.asset)
  ○ 元素       → 创建 ElementConfig (.asset)

→ 自动弹出 SaveFilePanel（地形例外，直接在 MapResourceConfig 中添加）
→ 填写表单 → 确认创建
```

#### 操作 B：拖入预制体导入（核心功能）

```
用户从 Project 窗口拖入一个或多个 prefab 到面板
  ↓
系统检测 prefab 类型：
  - 如果文件名匹配 *Unit* / *Enemy* / *NPC* → 建议类型 = 单位
  - 如果文件名匹配 *Tree* / *Rock* / *Prop* / *Building* → 建议类型 = 元素
  - 其他 → 建议类型 = 元素（可手动改为单位）

每个 prefab：
  1. 自动提取名称作为 ID（如 "Oak_Tree" → "oak_tree"）
  2. 提取名称作为显示名称（如 "Oak Tree" → "橡树" 或保持原名）
  3. 自动关联 prefab 引用
  4. 创建对应的配置资产

→ 弹窗确认: "检测到 N 个预制体，将创建为 [元素] 类型。确认？"
→ 批量创建完成 → 提示 "已创建 N 个配置"
→ 刷新列表，新配置可立即在地图上使用
```

#### 操作 C：扫描已存在资产

```
点击 [扫描] →
  扫描项目中所有:
    - MapResourceConfig (找 terrainResources)
    - UnitConfig (.asset)
    - ElementConfig (.asset)
  → 汇总到统一列表中显示
```

#### 操作 D：编辑资产

```
左侧列表选中一项 → 右侧显示详情:
  - 通用字段: ID, 名称, prefab, sprite
  - 类型特有字段:
    地形: 通行规则, 阻挡单位, 自定义属性
    单位: 攻击, 移速, 生命
    元素: 放置规则, 碰撞, 阻挡移动
  → 修改后点 [保存]
```

### 5.4 实现代码框架

```csharp
// Assets/Editor/MapEditor/MapAssetManagerWindow.cs

using System.Collections.Generic;
using System.Linq;
using UnityEditor;
using UnityEngine;

/// <summary>统一资产管理窗口</summary>
public class MapAssetManagerWindow : EditorWindow
{
    // ===== 数据 =====
    private MapResourceConfig _config;                          // 地形配置表
    private List<UnitConfig> _unitConfigs = new List<UnitConfig>();
    private List<ElementConfig> _elementConfigs = new List<ElementConfig>();

    // ===== UI 状态 =====
    private enum AssetTab { Terrain, Unit, Element }
    private AssetTab _currentTab = AssetTab.Terrain;

    private int _selectedTerrainIndex = -1;
    private int _selectedUnitIndex = -1;
    private int _selectedElementIndex = -1;

    private Vector2 _listScroll, _detailScroll;

    // 新建表单
    private string _newTerrainId, _newTerrainName;
    private DefaultTerrainType _newTerrainType;
    private Sprite _newTerrainSprite;
    private GameObject _newTerrainPrefab;

    private string _newUnitId, _newUnitName;
    private GameObject _newUnitPrefab;

    private string _newElementId, _newElementName;
    private GameObject _newElementPrefab;
    private ElementPlaceRule _newElementPlaceRule;

    // ===== 菜单 =====
    [MenuItem("地图编辑器/资产管理器")]
    public static void Open()
    {
        var window = GetWindow<MapAssetManagerWindow>("资产管理器");
        window.minSize = new Vector2(700, 500);
    }

    // ===== 主 OnGUI =====
    private void OnGUI()
    {
        DrawToolbar();
        EditorGUILayout.BeginHorizontal();

        // 左侧：列表
        EditorGUILayout.BeginVertical(GUILayout.Width(200));
        DrawAssetList();
        EditorGUILayout.EndVertical();

        // 右侧：详情
        EditorGUILayout.BeginVertical();
        DrawDetailPanel();
        EditorGUILayout.EndVertical();

        EditorGUILayout.EndHorizontal();
        HandleDragDrop();
    }

    // === 标签页工具栏 ===
    private void DrawToolbar()
    {
        EditorGUILayout.BeginHorizontal(EditorStyles.toolbar);
        _config = EditorGUILayout.ObjectField(_config, typeof(MapResourceConfig), false,
            GUILayout.Width(200)) as MapResourceConfig;

        GUILayout.FlexibleSpace();

        // 标签按钮
        if (GUILayout.Toggle(_currentTab == AssetTab.Terrain, "地形", EditorStyles.toolbarButton))
            _currentTab = AssetTab.Terrain;
        if (GUILayout.Toggle(_currentTab == AssetTab.Unit, "单位", EditorStyles.toolbarButton))
            _currentTab = AssetTab.Unit;
        if (GUILayout.Toggle(_currentTab == AssetTab.Element, "元素", EditorStyles.toolbarButton))
            _currentTab = AssetTab.Element;

        GUILayout.FlexibleSpace();

        if (GUILayout.Button("+ 新建", EditorStyles.toolbarButton, GUILayout.Width(60)))
            ShowCreateDialog();

        if (GUILayout.Button("扫描", EditorStyles.toolbarButton, GUILayout.Width(50)))
            ScanAllAssets();

        EditorGUILayout.EndHorizontal();
    }

    // === 资产列表 ===
    private void DrawAssetList()
    {
        _listScroll = EditorGUILayout.BeginScrollView(_listScroll);

        switch (_currentTab)
        {
            case AssetTab.Terrain:
                DrawTerrainList();
                break;
            case AssetTab.Unit:
                DrawUnitList();
                break;
            case AssetTab.Element:
                DrawElementList();
                break;
        }

        // 拖入提示
        EditorGUILayout.HelpBox("拖入 Prefab 到此区域\n自动创建配置", MessageType.Info);

        EditorGUILayout.EndScrollView();
    }

    private void DrawTerrainList()
    {
        if (_config?.terrainResources == null) return;
        for (int i = 0; i < _config.terrainResources.Count; i++)
        {
            var t = _config.terrainResources[i];
            if (t == null) continue;

            bool selected = _selectedTerrainIndex == i;
            GUI.backgroundColor = selected ? new Color(0.5f, 0.8f, 1f) : Color.white;
            if (GUILayout.Button($"📁 {t.terrainName} [{t.terrainId}]", GUILayout.Height(24)))
            {
                _selectedTerrainIndex = i;
                _selectedUnitIndex = -1;
                _selectedElementIndex = -1;
            }
        }
        GUI.backgroundColor = Color.white;
    }

    // DrawUnitList(), DrawElementList() 类似...

    // === 详情面板 ===
    private void DrawDetailPanel()
    {
        _detailScroll = EditorGUILayout.BeginScrollView(_detailScroll);

        switch (_currentTab)
        {
            case AssetTab.Terrain:
                DrawTerrainDetail();
                break;
            case AssetTab.Unit:
                DrawUnitDetail();
                break;
            case AssetTab.Element:
                DrawElementDetail();
                break;
        }

        EditorGUILayout.EndScrollView();
    }

    private void DrawTerrainDetail()
    {
        if (_selectedTerrainIndex < 0 ||
            _selectedTerrainIndex >= (_config?.terrainResources?.Count ?? 0))
        {
            DrawCreateTerrainForm();
            return;
        }

        var t = _config.terrainResources[_selectedTerrainIndex];

        EditorGUILayout.LabelField("地形详情", EditorStyles.boldLabel);
        t.terrainId = EditorGUILayout.TextField("唯一 ID", t.terrainId);
        t.terrainName = EditorGUILayout.TextField("显示名称", t.terrainName);

        // ===== 新增：预制体字段 =====
        EditorGUILayout.LabelField("可视化资源", EditorStyles.boldLabel);
        t.terrainSprite = EditorGUILayout.ObjectField("2D 精灵", t.terrainSprite, typeof(Sprite), false) as Sprite;
        t.prefab = EditorGUILayout.ObjectField("3D 预制体 (可选)", t.prefab, typeof(GameObject), false) as GameObject;

        if (t.prefab != null)
        {
            t.prefabOffset = EditorGUILayout.Vector3Field("位置偏移", t.prefabOffset);
            t.prefabRotation = EditorGUILayout.Vector3Field("旋转", t.prefabRotation);
            t.prefabScale = EditorGUILayout.FloatField("缩放", t.prefabScale);
        }

        t.defaultType = (DefaultTerrainType)EditorGUILayout.EnumPopup("通行规则", t.defaultType);

        // ... 现有自定义属性编辑（不变）...

        if (GUILayout.Button("应用修改", GUILayout.Height(30)))
            EditorUtility.SetDirty(_config);
    }

    private void DrawCreateTerrainForm()
    {
        EditorGUILayout.LabelField("创建新地形", EditorStyles.boldLabel);

        _newTerrainId = EditorGUILayout.TextField("地形 ID", _newTerrainId);
        _newTerrainName = EditorGUILayout.TextField("显示名称", _newTerrainName);
        _newTerrainType = (DefaultTerrainType)EditorGUILayout.EnumPopup("通行规则", _newTerrainType);
        _newTerrainSprite = EditorGUILayout.ObjectField("精灵", _newTerrainSprite, typeof(Sprite), false) as Sprite;

        // ===== 新增：预制体字段 =====
        EditorGUILayout.Space(5);
        EditorGUILayout.LabelField("快速导入", EditorStyles.boldLabel);
        _newTerrainPrefab = EditorGUILayout.ObjectField("从预制体导入", _newTerrainPrefab,
            typeof(GameObject), false) as GameObject;

        if (_newTerrainPrefab != null)
        {
            // 自动填充 ID 和名称
            if (string.IsNullOrEmpty(_newTerrainId))
                _newTerrainId = _newTerrainPrefab.name.ToLower().Replace(" ", "_");
            if (string.IsNullOrEmpty(_newTerrainName))
                _newTerrainName = _newTerrainPrefab.name;

            // 尝试从 prefab 中提取 SpriteRenderer 的 sprite
            var sr = _newTerrainPrefab.GetComponentInChildren<SpriteRenderer>();
            if (sr != null && _newTerrainSprite == null)
                _newTerrainSprite = sr.sprite;
        }

        if (GUILayout.Button("确认创建", GUILayout.Height(30)))
        {
            if (string.IsNullOrWhiteSpace(_newTerrainId))
            {
                EditorUtility.DisplayDialog("创建失败", "地形 ID 不能为空", "确定");
                return;
            }
            if (_config.ContainsId(_newTerrainId))
            {
                EditorUtility.DisplayDialog("创建失败", $"ID \"{_newTerrainId}\" 已存在", "确定");
                return;
            }

            var terrain = new TerrainResource
            {
                terrainId = _newTerrainId.Trim(),
                terrainName = string.IsNullOrWhiteSpace(_newTerrainName) ? _newTerrainId.Trim() : _newTerrainName.Trim(),
                defaultType = _newTerrainType,
                terrainSprite = _newTerrainSprite,
                prefab = _newTerrainPrefab   // ← 新字段
            };
            _config.terrainResources.Add(terrain);
            EditorUtility.SetDirty(_config);
            _selectedTerrainIndex = _config.terrainResources.Count - 1;
            ResetTerrainForm();
        }
    }

    private void DrawUnitDetail() { /* 类似地形详情 */ }
    private void DrawElementDetail() { /* 类似，但字段为 ElementPlaceRule 等 */ }

    // ===== 拖拽处理（核心） =====
    private void HandleDragDrop()
    {
        Event evt = Event.current;
        Rect dropArea = GUILayoutUtility.GetLastRect(); // 列表区域

        switch (evt.type)
        {
            case EventType.DragUpdated:
            case EventType.DragPerform:
                if (!dropArea.Contains(evt.mousePosition)) return;

                DragAndDrop.visualMode = DragAndDropVisualMode.Copy;

                if (evt.type == EventType.DragPerform)
                {
                    DragAndDrop.AcceptDrag();

                    var prefabs = DragAndDrop.objectReferences
                        .OfType<GameObject>()
                        .Where(g => PrefabUtility.GetPrefabAssetType(g) != PrefabAssetType.NotAPrefab)
                        .ToList();

                    if (prefabs.Count == 0)
                    {
                        EditorUtility.DisplayDialog("导入失败", "请拖入 Prefab 预制体", "确定");
                        return;
                    }

                    ImportPrefabs(prefabs);
                }
                evt.Use();
                break;
        }
    }

    /// <summary>导入预制体：根据当前标签页自动创建对应类型配置</summary>
    private void ImportPrefabs(List<GameObject> prefabs)
    {
        // 确认对话框
        string typeLabel = _currentTab switch
        {
            AssetTab.Terrain => "地形",
            AssetTab.Unit => "单位",
            AssetTab.Element => "元素",
            _ => "元素"
        };

        string msg = prefabs.Count == 1
            ? $"将 \"{prefabs[0].name}\" 导入为 [{typeLabel}] 配置？"
            : $"将 {prefabs.Count} 个预制体批量导入为 [{typeLabel}] 配置？\n\n" +
              string.Join("\n", prefabs.Select(p => $"  · {p.name}"));

        if (!EditorUtility.DisplayDialog("确认导入", msg, "确认", "取消"))
            return;

        int created = 0;
        foreach (var prefab in prefabs)
        {
            if (_currentTab == AssetTab.Terrain)
                created += ImportAsTerrain(prefab) ? 1 : 0;
            else if (_currentTab == AssetTab.Unit)
                created += ImportAsUnit(prefab) ? 1 : 0;
            else
                created += ImportAsElement(prefab) ? 1 : 0;
        }

        ScanAllAssets();
        Repaint();

        EditorUtility.DisplayDialog("导入完成", $"成功创建 {created}/{prefabs.Count} 个配置", "确定");
    }

    /// <summary>将 prefab 导入为地形资源</summary>
    private bool ImportAsTerrain(GameObject prefab)
    {
        if (_config == null)
        {
            Debug.LogError("请先关联地形资源配置表");
            return false;
        }

        string id = prefab.name.ToLower().Replace(" ", "_");
        if (_config.ContainsId(id))
        {
            Debug.LogWarning($"地形 ID \"{id}\" 已存在，跳过");
            return false;
        }

        var terrain = new TerrainResource
        {
            terrainId = id,
            terrainName = prefab.name,
            prefab = prefab,
            defaultType = DefaultTerrainType.Passable
        };

        // 尝试提取 sprite
        var sr = prefab.GetComponentInChildren<SpriteRenderer>();
        if (sr != null) terrain.terrainSprite = sr.sprite;

        _config.terrainResources.Add(terrain);
        EditorUtility.SetDirty(_config);
        return true;
    }

    /// <summary>将 prefab 导入为单位配置</summary>
    private bool ImportAsUnit(GameObject prefab)
    {
        string id = prefab.name.ToLower().Replace(" ", "_");

        // 检查重复
        var existing = AssetDatabase.FindAssets($"t:UnitConfig {id}")
            .Select(g => AssetDatabase.LoadAssetAtPath<UnitConfig>(AssetDatabase.GUIDToAssetPath(g)))
            .FirstOrDefault(c => c != null && c.unitId == id);
        if (existing != null)
        {
            Debug.LogWarning($"单位 ID \"{id}\" 已存在，跳过");
            return false;
        }

        // 保存到 Assets/ 下
        string path = EditorUtility.SaveFilePanelInProject(
            "保存单位配置", $"{prefab.name}_Config", "asset",
            "选择单位配置保存位置");

        if (string.IsNullOrEmpty(path)) return false;

        var unitConfig = CreateInstance<UnitConfig>();
        unitConfig.unitId = id;
        unitConfig.unitName = prefab.name;
        unitConfig.prefab = prefab;

        AssetDatabase.CreateAsset(unitConfig, path);
        AssetDatabase.SaveAssets();
        return true;
    }

    /// <summary>将 prefab 导入为元素配置</summary>
    private bool ImportAsElement(GameObject prefab)
    {
        string id = prefab.name.ToLower().Replace(" ", "_");

        var existing = AssetDatabase.FindAssets($"t:ElementConfig {id}")
            .Select(g => AssetDatabase.LoadAssetAtPath<ElementConfig>(AssetDatabase.GUIDToAssetPath(g)))
            .FirstOrDefault(c => c != null && c.elementId == id);
        if (existing != null)
        {
            Debug.LogWarning($"元素 ID \"{id}\" 已存在，跳过");
            return false;
        }

        string path = EditorUtility.SaveFilePanelInProject(
            "保存元素配置", $"{prefab.name}_Config", "asset",
            "选择元素配置保存位置");

        if (string.IsNullOrEmpty(path)) return false;

        var elementConfig = CreateInstance<ElementConfig>();
        elementConfig.elementId = id;
        elementConfig.elementName = prefab.name;
        elementConfig.prefab = prefab;

        // 尝试从 prefab 提取 sprite

        if (sr != null) elementConfig.previewSprite = sr.sprite;

        AssetDatabase.CreateAsset(elementConfig, path);
        AssetDatabase.SaveAssets();
        return true;
    }
```

### 6.2 预制体预览实现

```csharp
/// <summary>使用 Handles 绘制预制体预览</summary>
private static Dictionary<GameObject, Mesh> _prefabMeshCache = new Dictionary<GameObject, Mesh>();

private void DrawPrefabPreview(GameObject prefab, Vector3 position, TerrainResource terrain)
{
    // 从缓存获取或构建合并 Mesh
    if (!_prefabMeshCache.TryGetValue(prefab, out Mesh mesh))
    {
        mesh = BuildPreviewMeshFromPrefab(prefab);
        _prefabMeshCache[prefab] = mesh;
    }
    if (mesh == null) return;

    Vector3 pos = position + terrain.prefabOffset;
    Quaternion rot = Quaternion.Euler(terrain.prefabRotation);
    Vector3 scale = Vector3.one * terrain.prefabScale;

    // 根据通行规则染色
    Color col = GetTerrainColor(terrain);
    Handles.color = new Color(col.r, col.g, col.b, 0.5f);

    Handles.DrawMesh(mesh, pos, rot, scale);
}

/// <summary>从 prefab 提取 Mesh（合并所有 MeshFilter）</summary>
private static Mesh BuildPreviewMeshFromPrefab(GameObject prefab)
{
    var meshFilters = prefab.GetComponentsInChildren<MeshFilter>();
    if (meshFilters.Length == 0) return null;

    var combineInstances = new CombineInstance[meshFilters.Length];
    for (int i = 0; i < meshFilters.Length; i++)
    {
        combineInstances[i].mesh = meshFilters[i].sharedMesh;
        combineInstances[i].transform = meshFilters[i].transform.localToWorldMatrix;
    }

    var combined = new Mesh();
    combined.CombineMeshes(combineInstances);
    return combined;
}
```

**性能注意**：`_prefabMeshCache` 在编辑器生命周期内缓存，切换场景或脚本重编译时自动清空（静态字典）。

### 6.4 2D 模式预制体渲染 — 核心设计

> **关键问题**：Unity 2D Tilemap 只能放置 Tile（精灵），无法直接使用 GameObject prefab。而 2D 模式又是俯视角策略游戏的主要编辑模式。需要为预制体提供专门的 2D 渲染方案。

#### 6.4.1 总策略：三种资产类型，三种渲染方式

| 资产类型 | sprite 存在时 | prefab 存在时 | 两者都有时 |
|---------|-------------|-------------|-----------|
| **地形** | Tilemap tile（精灵） | 从 prefab 提取 sprite → Tilemap tile；3D prefab 无精灵则用色块标记 | Tilemap tile（优先 sprite） |
| **单位** | 无 sprite 字段 | 实例化 editor-only GameObject | 实例化 prefab |
| **元素** | previewSprite（可选） | 实例化 editor-only GameObject | 实例化 prefab |

#### 6.4.2 方案：editor-only 预制体实例化

在 `MapEditor2DSceneManager` 中维护一组临时实例化的 GameObject：

```csharp
// MapEditor2DSceneManager.cs 新增字段
private List<GameObject> _prefabInstances = new List<GameObject>();  // 临时实例化列表
private bool _show2DPrefabs = true;  // 用户可开关

/// <summary>同步 2D 预制体实例（编辑数据变更后调用）</summary>
public void SyncPrefabInstances()
{
    ClearPrefabInstances();

    if (!_show2DPrefabs) return;

    // 1. 实例化单位层预制体
    var unitLayers = _owner.AllUnitLayers;
    for (int layer = 0; layer < unitLayers.Count; layer++)
    {
        foreach (var kv in unitLayers[layer])
        {
            var unitConfig = _owner.GetUnitConfigById(kv.Value);
            if (unitConfig?.prefab == null) continue;

            Vector3 pos = HexToWorld ?
                HexGridUtils.HexToWorld2D(kv.Key, cs, orientation) :
                new Vector3(kv.Key.x * cs + cs * 0.5f, kv.Key.y * cs + cs * 0.5f, 0);
            pos.z = -0.1f * (layer + 1); // 分层 Z 排序

            InstanceForPreview(unitConfig.prefab, pos, Quaternion.identity,
                $"UnitPreview_{kv.Value}_{kv.Key.x}_{kv.Key.y}",
                new Color(0.5f, 0.5f, 1f, 0.7f)); // 蓝色半透明
        }
    }

    // 2. 实例化元素层预制体
    var elementLayers = _owner.AllElementLayers;
    for (int layer = 0; layer < elementLayers.Count; layer++)
    {
        foreach (var kv in elementLayers[layer])
        {
            var elementConfig = _owner.GetElementConfigById(kv.Value);
            if (elementConfig?.prefab == null) continue;

            Vector3 pos = HexToWorld ?
                HexGridUtils.HexToWorld2D(kv.Key, cs, orientation) :
                new Vector3(kv.Key.x * cs + cs * 0.5f, kv.Key.y * cs + cs * 0.5f, 0);
            pos.z = -0.1f * (layer + 1);

            InstanceForPreview(elementConfig.prefab, pos, Quaternion.identity,
                $"ElemPreview_{kv.Value}_{kv.Key.x}_{kv.Key.y}",
                Color.white); // 保持原色
        }
    }
}

/// <summary>实例化一个编辑器预览用的 GameObject</summary>
private void InstanceForPreview(GameObject prefab, Vector3 position,
    Quaternion rotation, string name, Color tint)
{
    var instance = (GameObject)PrefabUtility.InstantiatePrefab(prefab);
    instance.transform.SetParent(_container.transform);
    instance.transform.position = position;
    instance.transform.rotation = rotation;
    instance.name = name;
    instance.hideFlags = HideFlags.DontSave; // 不保存在场景中

    // 染色：给所有 Renderer 加半透明效果
    foreach (var renderer in instance.GetComponentsInChildren<Renderer>())
    {
        renderer.sharedMaterial = new Material(renderer.sharedMaterial);
        renderer.sharedMaterial.color = tint;
        // 如果在 2D 模式下用 SpriteRenderer
        if (renderer is SpriteRenderer sr)
        {
            sr.sortingOrder = -1;
        }
    }

    _prefabInstances.Add(instance);
}

/// <summary>清除所有临时实例</summary>
public void ClearPrefabInstances()
{
    foreach (var go in _prefabInstances)
    {
        if (go != null)
            DestroyImmediate(go);
    }
    _prefabInstances.Clear();
}
```

## 十六、核心设计原则总结

1. **一个面板管所有**：`MapAssetManagerWindow` 统一管理三种资产类型，拖入 prefab 即可导入
2. **预制体驱动**：TerrainResource + UnitConfig + ElementConfig 全部支持 prefab，编辑器可选择显示实际模型
3. **元素层独立**：新增 `LayerType.Element`，与地形/单位层平行管理，共享撤销/重做/序列化体系
4. **最小侵入**：TerrainResource 仅增加 prefab 字段（可选），旧配置完全兼容
5. **统一序列化**：`MapJsonData` 新增 `elementDatas` 字段，与 `cellDatas`/`unitDatas` 平行存储
6. **批量场景适配**：右键菜单 + 文件夹导入 + 拖拽面板，满足从单个到数百个 prefab 的导入需求

### P23：ElementPlaceRule 定义了但从未被校验

**问题**：`ElementPlaceRule` 枚举有 4 种规则（Free/RequiresTerrain/EmptyOnly/Unique），但在 `PaintElementCell()` 中没有对这些规则做任何校验。用户放置元素时不会受到规则约束。

**补充方案**：

```csharp
/// <summary>判断是否可以在指定格子放置元素</summary>
private bool CanPlaceElement(Vector2Int cellPos, string elementId)
{
    var elementConfig = GetElementConfigById(elementId);
    if (elementConfig == null) return false;

    switch (elementConfig.placeRule)
    {
        case ElementPlaceRule.Free:
            return true;

        case ElementPlaceRule.RequiresTerrain:
            // 必须已有地形
            return _mapGrid.ContainsKey(cellPos);

        case ElementPlaceRule.EmptyOnly:
            // 必须没有任何物件（地形、单位、元素）
            return !_mapGrid.ContainsKey(cellPos) &&
                   !_unitGrid.ContainsKey(cellPos) &&
                   !_elementGrid.ContainsKey(cellPos);

        case ElementPlaceRule.Unique:
            // 全局唯一：检查所有元素层是否已存在该 ID
            foreach (var layer in _allElementLayers)
                if (layer.ContainsValue(elementId))
                    return false;
            return true;

        default:
            return true;
    }
}

// 修改 PaintElementCell():
private void PaintElementCell(Vector2Int cellPos, EditMode mode)
{
    if (mode == EditMode.Brush)
    {
        if (_selectElementIndex < _elementConfigs.Count)
        {
            string eid = _elementConfigs[_selectElementIndex].elementId;
            if (!CanPlaceElement(cellPos, eid))
            {
                Debug.LogWarning($"无法在此位置放置元素 {eid}，不满足放置规则");
                return;
            }
            RecordChange(cellPos);
            _elementGrid[cellPos] = eid;
        }
    }
    // ...
}
```

### P29：运行时地形 prefab 实例化缺失

**问题**：`RuntimeMapGenerator.SpawnCell()` 只创建 `SpriteRenderer`，如果地形有 prefab，运行时应该实例化 prefab 而不是创建 sprite。

**补充方案**：

```csharp
private void SpawnCell(MapCellData cell)
{
    var terrain = config.GetTerrainById(cell.terrainId);
    if (terrain == null) return;

    Vector3 pos = new Vector3(cell.x + 0.5f, spriteYOffset, cell.y + 0.5f);
    GameObject cellGo;

    // prefab 优先
    if (terrain.prefab != null)
    {
        cellGo = Instantiate(terrain.prefab, pos,
            Quaternion.Euler(terrain.prefabRotation), mapRoot);
        cellGo.transform.localScale *= terrain.prefabScale;
        cellGo.name = $"Cell_{cell.x}_{cell.y}";
    }
    else
    {
        // 无 prefab：回退到 SpriteRenderer（现有逻辑）
        cellGo = new GameObject($"Cell_{cell.x}_{cell.y}");
        cellGo.transform.SetParent(mapRoot);
        cellGo.transform.position = pos;

        if (terrain.terrainSprite != null)
        {
            var sr = cellGo.AddComponent<SpriteRenderer>();
            sr.sprite = terrain.terrainSprite;
            sr.color = terrain.defaultType switch
            {
                DefaultTerrainType.Passable => new Color(1f, 1f, 1f, 0.9f),
                DefaultTerrainType.Impassable => new Color(1f, 0.7f, 0.7f, 0.9f),
                DefaultTerrainType.UnitImpassable => new Color(1f, 1f, 0.7f, 0.9f),
                _ => Color.white
            };
        }
    }

    // 碰撞（一致）
    if (generateColliders && terrain.defaultType == DefaultTerrainType.Impassable)
    {
        // 如果 prefab 已有 Collider，则不额外添加
        if (cellGo.GetComponent<Collider>() == null)
            cellGo.AddComponent<BoxCollider>();
    }

    var cellData = cellGo.AddComponent<TerrainCellData>();
    cellData.terrainId = cell.terrainId;
```

### P35：三层数组需要在所有层操作中保持同步

**问题**：添加/删除层时，`_allLayers`、`_allUnitLayers`、`_allElementLayers` 的数组长度必须始终一致（通过 `_layerTypes` 区分每层的实际类型）。

**补充方案**：

```csharp
// 添加层的统一方法
private void AddLayer(LayerType type)
{
    _allLayers.Add(new Dictionary<Vector2Int, string>());
    _allUnitLayers.Add(new Dictionary<Vector2Int, string>());
    _allElementLayers.Add(new Dictionary<Vector2Int, string>());
    _layerTypes.Add(type);
    _currentLayer = _layerTypes.Count - 1;
}

// 删除层的统一方法
private void RemoveLayer(int index)
{
    if (_layerTypes.Count <= 1) return; // 至少保留一层

    _allLayers.RemoveAt(index);
    _allUnitLayers.RemoveAt(index);
    _allElementLayers.RemoveAt(index);
    _layerTypes.RemoveAt(index);

    _currentLayer = Mathf.Min(_currentLayer, _layerTypes.Count - 1);
}

// 访问当前层的实际数据：
private Dictionary<Vector2Int, string> _activeGrid
{
    get
    {
        return _layerTypes[_currentLayer] switch
        {
            LayerType.Terrain => _allLayers[_currentLayer],
            LayerType.Unit => _allUnitLayers[_currentLayer],
            LayerType.Element => _allElementLayers[_currentLayer],
            _ => _allLayers[_currentLayer]
        };
    }
}
```

### P39：PrefabInstantiate 材质修改可能污染原资产

**问题**：6.4.2 节 `InstanceForPreview` 中直接修改 `renderer.sharedMaterial.color`。`sharedMaterial` 修改会影响到所有使用该材质的对象。应该创建临时材质副本。

```csharp
    // ...

    {
        // 创建临时材质副本（不影响原 prefab）
        var tempMat = new Material(renderer.sharedMaterial);
        tempMat.color = new Color(
            tempMat.color.r * tint.r,
            tempMat.color.g * tint.g,
            tempMat.color.b * tint.b,
```
