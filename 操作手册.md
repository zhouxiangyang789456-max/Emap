# Unity 地形地图编辑器 — 操作手册

本手册假设你**从未用过 Unity 编辑器**，每一步都会截图式描述（用文字描述界面，你可以对照操作）。

---

## 第一部分：安装与导入（2 分钟）

### 步骤1：确认 Unity 版本

打开 Unity Hub，确认你的 Unity 版本是 **2020.3 或更高**。
- 如果低于 2020.3：去 Unity Hub → Installs → Install Editor → 选 2020.3 LTS

### 步骤2：创建或打开项目

- **新建项目**：Unity Hub → New Project → 选 "3D Core" → 起个名字如 `MyMapGame` → Create
- **已有项目**：直接打开

### 步骤3：导入插件文件

1. 打开本文件夹，找到 `Assets` 文件夹
2. 打开你 Unity 项目的文件夹（在资源管理器中）
3. 把本插件的 `Assets` 文件夹**拖入**项目根目录
4. 如果提示覆盖，选"是"
5. 回到 Unity，等待右下角进度条跑完（约 10-30 秒）

---

## 第二部分：创建地形配置表（3 分钟）

> 地形配置表 = 一张表，列出你游戏里所有地形类型（草地、石头、水域等）

### 步骤4：创建配置文件

1. 在 Unity 下方 **Project 窗口**（文件浏览器）中，找到 `Assets` 文件夹
2. 右键空白处 → **Create** → **地图编辑器** → **地形资源配置表**
3. 出现一个文件，改名为 `MapConfig`（或保留默认名）
4. **单击选中它**，右侧 Inspector（属性面板）会显示它的内容

### 步骤5：添加第一个地形——"草地"

1. 在 Inspector 中看到 `Terrain Resources` 列表（当前为空）
2. 点击列表下方的 **"+ 添加新地形"** 按钮
3. 会展开一个编辑区域，按以下填写：

```
唯一 ID:        grass
显示名称:       草地
预览图标:       (可选，可以拖入一张图片)
通行规则:       Passable（可通过）
```

4. 找到"自定义属性"区域，点 **"+ 添加自定义属性"**，填写：

```
属性名:          攻击加成
属性类型:        Int
数值:            1
效果描述:        站在草地攻击力+1
```

5. 再点 **"+ 添加自定义属性"**，填写：

```
属性名:          移速加成
属性类型:        Float
数值:            0.2
效果描述:        移速提升20%
```

### 步骤6：添加第二个地形——"山石"

1. 再点 **"+ 添加新地形"**
2. 填写：

```
唯一 ID:        stone
显示名称:       山石
通行规则:       Impassable（不可通过）
```

3. **不需要自定义属性**，石头就是挡路的

### 步骤7：添加第三个地形——"浅滩"

1. 再点 **"+ 添加新地形"**
2. 填写：

```
唯一 ID:        shoal
显示名称:       浅滩
通行规则:       UnitImpassable（部分单位不可通过）
```

3. 在"禁止通行的单位 ID 列表"中点 **"+ 添加禁止单位 ID"**，输入 `infantry`
4. 添加自定义属性：

```
属性名:          是否中毒地形
属性类型:        Bool
是否启用:        ✓（勾上）
效果描述:        站在浅滩持续中毒掉血
```

5. 至此，你有了 3 个地形：草地、山石、浅滩

---

## 第三部分：打开地图编辑器（1 分钟）

### 步骤8：打开编辑器窗口

1. 点击 Unity 顶部菜单栏 → **地图编辑器** → **打开地图编辑器**
2. 一个名为"地形地图编辑器"的窗口会弹出
3. **将窗口拖到 Scene 视图旁边**，方便同时看编辑器和场景

### 步骤9：关联配置表

1. 在编辑器窗口最上方，有一个"地形资源配置表"字段（显示为 None）
2. 从 Project 窗口**拖拽**刚才创建的 `MapConfig` 文件到这个字段上
3. 编辑器下方会显示你的 3 个地形

---

## 第四部分：编辑你的第一张地图（5 分钟）

### 步骤10：设置地图大小

在编辑器窗口找到"地图基础参数"区域：

```
地图名称:        MyFirstMap
地图宽度:        30（30 个格子宽）
地图高度:        20（20 个格子高）
格子尺寸:        1（每个格子 1×1 单位）
```

### 步骤11：认识 Scene 视图

现在看 **Scene 视图**（中间最大的那个 3D 窗口）：

- 显示了一个**绿色网格**，这是你的地图编辑区
- **原点 (0,0)** 处有一个**青色箭头标记**
- 左下角是 (0,0)，X 轴向右，Y 轴（在 3D 里是 Z 轴）向前

**Scene 视图操作**（不会的话先练一下）：
- **右键拖动**：旋转视角
- **滚轮**：缩放
- **中键拖动**：平移
- 调整到**俯视角度**（从上往下看），编辑最方便

### 步骤12：画第一块地形

1. 在编辑器窗口找到"选中地形"下拉框
2. 选择 `[grass] 草地`
3. 编辑模式选 **Brush（刷子）**
4. 把鼠标移到 Scene 视图的地图上
5. 你会看到鼠标旁有一个**半透明绿色方块**跟着鼠标移动
6. 鼠标旁还有一个提示框显示"🖌 草地 (grass) @ [x,y]"
7. **左键点击**网格中任意一个格子 → 刷上了！格子上出现一个绿色半透明色块

### 步骤13：画一片区域

1. **按住左键不放，拖动鼠标** → 可以连续铺设！走过的格子都会变成草地
2. 从 (0,0) 拖到 (10,5)，画出一片 10×5 的草地区域

### 步骤14：画阻挡地形

1. 地形下拉框切换到 `[stone] 山石`
2. 在草地中间点击几个格子 → 出现**红色色块**（表示不可通行）
3. 可以画一条石墙把地图分成两半

### 步骤15：画浅滩

1. 地形下拉框切换到 `[shoal] 浅滩`
2. 在地图边缘点几个格子 → 出现**黄色色块**（部分单位不可通行）

### 步骤16：擦除

两种方式擦除：

**方式A**：编辑模式切换到 **Eraser（橡皮）**，左键点击擦除

**方式B**：保持 Brush 模式，**按住 Shift 键**，左键点击即可临时切换为橡皮

### 步骤17：撤销 / 重做

- 按 **Ctrl+Z** → 撤销上一步
- 按 **Ctrl+Y** → 重做刚才撤销的
- 或者点击编辑器里的 **↩ 撤销** 和 **↪ 重做** 按钮
- 按钮上会显示 `撤销 (5)` 表示还可以撤销 5 步

## 第五部分：使用高级编辑工具（3 分钟）

### 步骤18：矩形填充

1. 编辑模式切换到 **RectangleFill（矩形填充）**
2. 选择 `[grass] 草地`
3. 在 Scene 视图中，**按住左键从一个角拖到对角**
4. 松开鼠标 → 整个矩形区域瞬间铺满草地
5. 适合快速填充大面积

### 步骤19：油漆桶（Flood Fill）

1. 编辑模式切换到 **FloodFill（油漆桶）**
2. 选择 `[stone] 山石`
3. 在 Scene 视图中，**点击一片连续草地的中央**
4. 整片相连的草地全部变成山石
5. 适合快速替换整块区域

### 步骤20：多层地图

1. 在编辑器"层级管理"区域，点击 **"+ 添加新层"**
2. 当前编辑层变为"层 1"
3. 在层 1 上画一些地形（比如天花板的藤蔓）
4. 点击 **"◀ 上层"** 切回层 0，你之前画的东西还在
5. 多层数据会一起保存到 JSON

---

## 第六部分：保存与加载（2 分钟）

### 步骤21：保存地图

1. 点击编辑器中的 **"保存地图为 JSON"** 按钮
2. 弹出文件保存对话框
3. 导航到 `Assets` 文件夹下
4. 文件名输入 `MyFirstMap`，点击保存
5. 弹出 "保存成功" 提示
6. 在 Project 窗口中可以看到 `MyFirstMap.json` 文件

### 步骤22：加载地图

1. 点击 **"从 JSON 加载地图"** 按钮
2. 选择刚才保存的 `MyFirstMap.json`
3. 如果有数据校验问题会弹出警告（比如有些地形 ID 在配置表中不存在）
4. 点击"仍然加载"继续
5. 地图恢复到你保存时的状态

### 步骤23：自动恢复

- 如果不小心**关闭了编辑器窗口**，重新打开（菜单 → 地图编辑器 → 打开地图编辑器）
- 窗口重新打开后，**之前的地图数据会自动恢复**
- 这是因为编辑器会自动保存到 SessionState

## 第七部分：运行时使用地图（5 分钟）

### 步骤24：让游戏加载地图

1. 在 Hierarchy 窗口中，右键 → Create Empty，起名为 `MapManager`
2. 在 Inspector 中点击 **Add Component** → 搜索 `RuntimeMapGenerator` → 添加
3. 设置组件：

```
Config:         拖动 MapConfig 文件到这里
Map Json:       拖动 MyFirstMap.json 到这里
Map Root:       留空（自动创建）
```

### 步骤25：初始化全局配置

创建初始化脚本 `GameInitializer.cs`：

1. 在 `Assets/Scripts` 右键 → Create → C# Script → 起名 `GameInitializer`
2. 双击打开，替换为以下内容：

```csharp
using UnityEngine;

public class GameInitializer : MonoBehaviour
{
    public MapResourceConfig config;

    private void Awake()
    {
        GameMapGlobal.MapConfig = config;
        Debug.Log("地图系统初始化完成");
    }
}
```

3. 在场景中新建空物体，挂上 `GameInitializer`，拖入 `MapConfig`

### 步骤26：创建玩家单位测试

1. Hierarchy 右键 → 3D Object → Capsule → 起名 `Player`
2. Add Component → 搜索 `UnitTerrainLogic` → 添加
3. 设置：

```
Unit Id:       cavalry
Unit Config:   右键 Create → 地图编辑器 → 单位配置，创建后拖入
```

### 步骤27：手动测试地形交互

在 `UnitTerrainLogic` 挂载的物体上，右键 → 添加测试脚本，或在代码中调用：

```csharp
// 测试走在草地上
GetComponent<UnitTerrainLogic>().OnEnterGrid("grass");
// 测试走在大石头上（会被阻挡）
GetComponent<UnitTerrainLogic>().OnEnterGrid("stone");
```

观察 Console 窗口的输出日志，验证：
- 草地显示"攻击 +1, 移速 +20%"
- 山石显示"无法通过"
- 浅滩对 cavalry 可通过但中毒

## 第八部分：寻路测试（3 分钟）

### 步骤28：测试 A* 寻路

创建测试脚本或在 Update 中调用：

```csharp
using UnityEngine;

public class PathTest : MonoBehaviour
{
    public TextAsset mapJson;

    private void Start()
    {
        var data = JsonUtility.FromJson<MapJsonData>(mapJson.text);

        // 从 (0,0) 寻路到 (10,10)
        var path = MapPathfinding.FindPath(0, 0, 10, 10, "knight", data);

        if (path.Count > 0)
        {
            Debug.Log($"找到路径，共 {path.Count} 步");
            foreach (var step in path)
                Debug.Log($"  → ({step.x}, {step.y})");
        }
        else
            Debug.Log("无可行路径");
    }
}
```

## 常见问题排查

### Q：编辑器窗口一片空白，没有地形列表？

**A**：你还没拖入配置表。从 Project 窗口拖 MapConfig 到编辑器的"地形资源配置表"字段。

### Q：Scene 视图看不到网格？

**A**：Scene 视图摄像机可能离太远。选中 Scene 视图，按 **F 键**（聚焦），调整视角。

### Q：点击格子没有反应？

**A**：检查：
1. 确保选了一个地形（下拉框不要为空）
2. 确保编辑模式是"Brush"而不是"Eraser"
3. 在 Scene 视图中操作，不是在 Game 视图中

### Q：保存的 JSON 加载后地形不对？

**A**：可能配置表中的地形 ID 被改过了。JSON 里存的是地形 ID（如 "grass"），加载时会去配置表查找。如果 ID 不匹配，该格会显示灰色。

### Q：按 Ctrl+Z 没反应？

**A**：确保焦点在编辑器窗口或 Scene 视图上（点击一下编辑器窗口标题栏）。

### Q：游戏运行时报错 "MapConfig 未初始化"？

**A**：你还没调用 `GameMapGlobal.MapConfig = yourConfig`。在游戏启动脚本中初始化。

### Q：大型地图（200×200）卡顿？

**A**：换成 `RuntimeTilemapGenerator` 替代 `RuntimeMapGenerator`。它在场景中创建 Tilemap 组件，性能好得多。

## 操作速查卡

| 操作 | 方法 |
|------|------|
| 画一个格子 | 选地形 → Brush → 左键点击 Scene 格子 |
| 连续画一片 | 按住左键拖拽 |
| 擦除一个格子 | 切换到 Eraser 模式，或按住 Shift + 左键点击 |
| 填充矩形 | 切换到 RectangleFill → 按住左键拖拽对角线 |
| 油漆桶 | 切换到 FloodFill → 点击一片连续区域 |
| 撤销 | Ctrl+Z |
| 重做 | Ctrl+Y |
| 切换层 | 编辑器层滑块 / "◀ 上层" "下层 ▶" |
| 添加新层 | "+ 添加新层" 按钮 |
| 保存 | "保存地图为 JSON" |
| 加载 | "从 JSON 加载地图" |
| 切换地形 | 地形下拉框 |
| 添加新地形 | 配置列表底部 "+ 添加新地形" |
